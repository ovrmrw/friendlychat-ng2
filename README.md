|Editor|VSCode (stable)|
|node --version|v6.3.1|
|@angular/core|2.0.0-rc.5|

### Setup
```
$ npm install
```

### Run
```
$ npm start
```

### Check whether your TypeScript codes are correct
```
$ npm run tscheck
```

### Unit Test
```
$ npm run karma
or
$ npm run karma:w
```

### Rxjs5 Marble Test
```
$ npm run testrxjs
or
$ npm run testrxjs:w
```

### Unit Test & Rxjs5 Marble Test at the same time
```
$ npm test
```

### Deploy to firebase hosting
```
$ npm run deploy
```
